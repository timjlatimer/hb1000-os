The Vault

"If it is not in the vault, it does not exist." — the shared data layer (Phase 2 · Slice 0)
🔒 Nothing auto-commits. Decisions made in the app live in this browser until you approve a push to the vault. Your GitHub token is stored in this browser only and is sent only to api.github.com. Restores overwrite local data — export first if unsure.

Setup — vault token

Create one at github.com/settings/personal-access-tokens/new: Repository access → Only select repositories → timjlatimer/hb1000-os · Permissions → Contents: Read and write · nothing else. Reading vault status works without a token; committing requires one.
